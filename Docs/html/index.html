The documentation of standalone GEMS3K code consists of HTML files located in the "GEMS3K/doc/html/" folder in the trunk GEMS3K subversion repository.

The recommended way to prepare GEMS3K input files consists in creating a modeling project in GEM-Selektor package, creating and calculating chemical systems (equilibria), and then exporting the GEM task records to GEMS3K I/O files.
